## What to tell buyers who ask "Spigot?"

ItemMagnet requires **Paper 1.21.1+**. It does not run on vanilla Spigot because it uses Paper API features and modern interact handling for fuel transfer. Paper is a drop-in replacement for Spigot on most hosts.
